BRAVE 是一个生物信息响应式分析与可视化引擎（Bioinformatics Reactive Analysis and Visualization Engine）。它打破了传统生信分析中环境配置复杂、流程维护困难的壁垒，实现了科研与工程的深度结合。在 BRAVE 的生态中，工程师负责开发、封装并发布高质量的生物信息上游与下游分析工具；而研究人员则能像安装手机 App 一样，轻松一键安装所需的分析流程，并享受后续的一键实时更新服务。通过这种创新的模式，BRAVE 让研究人员彻底摆脱繁琐的技术细节，将宝贵的精力专注于数据解读与科学发现本身。

## 快速安装
```
curl -s https://raw.githubusercontent.com/pybrave/brave/refs/heads/master/install.sh  | bash
```
+ <http://localhost:5000>


使用阿里云镜像
```
curl -s https://raw.githubusercontent.com/pybrave/brave/refs/heads/master/install.sh  | bash -s -- --aliyun
```

默认安装位置是 `$HOME/brave-install`，但你可以使用 `--base-dir` 参数指定安装位置。
```
curl -s https://raw.githubusercontent.com/pybrave/brave/refs/heads/master/install.sh  | bash -s -- --aliyun --base-dir /opt/brave
```

在中国大陆，你可以使用以下命令通过阿里云镜像安装 BRAVE：
```
curl -s https://raw.giteeusercontent.com/pybrave/brave/raw/master/install.sh  | bash -s -- --aliyun
```
